# About This Repo

这个仓库的目的是存储本人在学习过程中学到的或者自己写出的代码, 这些代码的价值是帮助日后回顾知识点, 以免总是在网上乱翻.

我希望这个仓库能够在我日后忘记某些API时为我提供帮助, 也就是起到个人代码片的存储作用.
